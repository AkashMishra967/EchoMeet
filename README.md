EchoMeet is a full-stack video conferencing web application inspired by Zoom.
It enables users to create and join virtual meeting rooms, communicate via video/audio, and interact in real-time.

This project showcases real-time communication, WebRTC integration, and scalable full-stack architecture.

Features

 Real-time Video Calling
 Audio Streaming (Mute/Unmute)
 Create & Join Meeting Rooms
 Unique Meeting Links
 Live Chat during meetings
 Multiple Participants Support
 Authentication (Login / Signup)
 Responsive UI (Mobile + Desktop)

 Tech Stack
 
Frontend

React.js
WebRTC
Socket.io (Client)

Backend

Node.js
Express.js
Socket.io (Server)

Database

MongoDB (if authentication stored)

Tools & Libraries

PeerJS / WebRTC APIs
JWT Authentication
Axios
